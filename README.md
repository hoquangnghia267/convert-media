# Image Converter Tool

A modern GUI tool built with Python to convert images between different formats (PNG, JPG, WEBP, BMP).

## Features
- Modern UI using `customtkinter`.
- Bulk conversion support.
- Handles transparency (converts RGBA to RGB when saving as JPG).
- Real-time logs and progress bar.

## Requirements
- Python 3.12+
- Packages listed in `requirements.txt`

## How to run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python main.py
   ```
